> **40% of Americans** live in areas with unhealthy air quality. Pollution varies dramatically block-by-block. Most people have no idea what's in the air around their home.

## The Solution

5 agents using free government APIs to give you hyperlocal environmental intelligence.

## Agents

**AirQualityMonitor** — Real-time AQI by ZIP via EPA AirNow API
**WaterQualityTracker** — Tap water contamination data via EWG/EPA
**CarbonCalculator** — Personal carbon footprint with reduction suggestions
**ExtremeWeatherAlert** — Severe weather monitoring and evacuation routes
**PolicyTracker** — Climate legislation tracking and local environmental initiatives

## Safety

No safety-critical decisions. Recommends consulting local authorities for evacuation decisions.

## Data Sources

EPA AirNow REST API (free), EPA TRI (Toxic Release Inventory), EWG databases, NOAA weather data
